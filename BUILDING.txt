Stuff the instructions assume you have:
    * NSIS (http://nsis.sourceforge.net/Main_Page)
    * AutoHotkey (http://www.autohotkey.com/)

Steps to build the installer:
    1) Create a folder to put all the necessary bits in.
    2) Download Meld (http://ftp.gnome.org/pub/GNOME/sources/meld/1.7/meld-1.7.0.tar.xz).
    3) Extract Meld into a folder named 'meld' in your new folder.
    4) Apply this patch to the 'bin\meld' file: http://git.gnome.org/browse/meld/commit/?id=c8e32f072e3a75132826b9ef52c2e51461f26e12.  This is necessary because otherwise Windows will launch a command prompt window in addition to the Meld application window.
    5) Download Portable Python (http://www.portablepython.com/wiki/PortablePython2.7.3.1).
    6) Run Portable Python installer to put Python and PyGTK (and nothing else) into a folder called 'python' in your new folder.
    7) Put all 'meld.exe' and 'meld.ico' into the 'meld' folder.
    8) Put 'installer.nsi' and 'LICENSES.rtf' into your new folder.
    9) Compile 'meld.ahk' with AutoHotkey.  Note that to set the icon on the generated .exe, you will need to comile through the GUI and not the Windows Explorer right click.
   10) Add the generated 'meld.exe' into the 'meld' folder.
   11) Compile 'installer.nsi' with NSIS.

Steps to build the portable zip archive:
    1) Follow steps 1-6 of the installer instructions
    2) Use 7-Zip or similar program to create the zip archive of the two folders.
